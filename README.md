# 🏋️ Fit Tech | Análisis de Comportamiento de Usuarios

## 📌 Descripción del proyecto

Se ha detectado en Fit.ly Tech, una aplicación fitnes, que los últimos meses ha incrementado el abandono de usuarios, en este proyecto se realizará el comportamiento de los usuarios, asi como causas potenciales del abandono con el objetivo de proponer plan de acción que puede ser aplicado en el primer trimestre y metricas para medir el éxito del plan.

---

## 🛠️ Herramientas utilizadas

* Python (Pandas, NumPy)
* Colab Notebook
* Matplotlib / Seaborn

---

## 📂 Estructura del proyecto

* `data/`: dataset de usuarios y actividad
* `notebooks/`: análisis exploratorio (EDA)
* `src/`: limpieza y transformación de datos

---

## 🔍 Proceso de análisis

1. Limpieza y validación de datos

   * Manejo de valores nulos
   * Consistencia entre datos
   * Buscar duplicados
   * Formato y tipo de dato
   * Cerificación de categorias
 2. Análisis exploratorio
    Se priorizó el análisis de las variables más relevantes para la empresa como el tipo de membresía, la interacción del usuario y el soporte técnico,            comparando el comportamiento de los usuarios activos frente a aquellos que abandonaron.
---

## 📈 Valor generado

Este análisis permite enfocar esfuerzos en etapas críticas del ciclo de vida del usuario y su satisfacción impactando directamente en retención y crecimiento del negocio.
